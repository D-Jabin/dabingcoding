# SpriteMaker 使用说明

## 一、软件简介

SpriteMaker 是一款专业的游戏精灵图集制作工具，帮助开发者将视频转换为帧序列，进行背景抠图处理，最后排版成 Sprite Sheet（精灵图/雪碧图）格式。

## 二、系统要求

### 硬件要求
- **操作系统**：Windows 10 64位及以上版本（Windows 10 1809+、Windows 11）
- **处理器**：Intel Core i5 或更高配置（推荐）
- **内存**：4GB 及以上（推荐）
- **磁盘空间**：2GB 可用空间（用于存放帧序列和输出文件）
- **显示器分辨率**：1080×600 及以上

## 三、启动软件

1. 打开软件所在文件夹 `SpriteMaker/`
2. 双击 `SpriteMaker.exe` 启动软件
3. 软件启动后会自动进入帧提取界面
4. 所有导入导出目录路径不要包括汉字符号

## 五、基本使用流程

SpriteMaker 包含三个主要功能模块，按顺序使用：

1. **帧提取**：将视频转换为帧序列
2. **背景抠图**：对帧序列进行背景去除处理
3. **精灵图排版**：将帧序列排版成 Sprite Sheet

### 步骤一：帧提取

1. **选择视频文件**
   - 点击「选择视频」按钮
   - 选择支持的视频格式（MP4、MOV、WebM、AVI、FLV 等）
   - 视频加载后会在预览区显示第一帧

2. **设置提取参数**
   - **提取 FPS**：设置每秒提取的帧数（范围 1-60），默认 5 FPS
   - **起始时间/结束时间**：设置提取的时间范围
   - **时间范围滑块**：拖动双滑块快速调整提取范围
   - **视频裁剪区域**：在预览区拖拽白色边框调整裁剪区域

3. **高级设置**
   - **智能去重**：自动去除连续的相似帧，减少帧数量
   - **跳变帧检测**：检测场景切换产生的不连续帧

4. **设置输出目录**
   - 点击「设置输出目录」按钮选择保存位置

5. **开始提取**
   - 点击「提取帧」按钮
   - 状态栏显示提取进度
   - 提取完成后会提示提取的帧数
   - 提取的帧文件名与视频文件名一致

### 步骤二：背景抠图

1. **导入帧序列**
   - 点击「导入帧序列」按钮
   - 选择包含帧图片的文件夹
   - 帧图片会显示在右侧缩略图列表中

2. **设置抠图参数**
   - **取色器**：点击激活后，在预览区点击获取背景色
   - **颜色容差**：控制背景色的识别范围，数值越大识别范围越广
   - **边缘羽化**：控制边缘的平滑过渡效果
   - **边缘平滑**：对边缘进行高斯模糊处理

3. **手动微调**
   - **画笔工具**：
     - 加选模式（涂白）：增加前景区域
     - 减选模式（涂黑）：减少前景区域
   - **智能分割**：基于当前选择的目标智能分割目标于背景去噪

4. **播放控制**
   - 播放/暂停：播放帧序列预览抠图效果
   - 速度调节：调整播放速度（0.5×-5×）
   - 上一张/下一张：手动切换帧

5. **对比原图**
   - 按住「对比原图」按钮显示原图，释放恢复抠图结果

6. **设置输出目录**
   - 点击「设置输出目录」按钮选择保存位置

7. **导出抠图结果**
   - 点击「导出全部」按钮
   - 状态栏显示导出进度
   - 提取出的抠图帧文件名与导出文件夹名一致

### 步骤三：精灵图排版

1. **选择输入目录**
   - 点击「选择输入目录」按钮
   - 选择包含处理后帧图片的文件夹
   - 输出目录会自动设置为输入目录下的 `/spritesheet` 文件夹

2. **选择排版模式**

   **模式A：现代透明裁剪图集（流式布局）**
   - 自动裁剪每帧的透明区域
   - 按最大图集宽度自动排版
   - 支持设置水平间距、垂直间距和边框大小
   - 适用于角色动画、特效等不规则帧尺寸

   **模式B：老式网格行列图集**
   - 所有帧统一尺寸，按网格排列
   - 需要设置列数
   - 适用于瓦片地图、UI 等需要严格对齐的场景

3. **设置布局参数**
   - **缩放尺寸**：按最长边缩放至指定尺寸（32px/64px/128px/自定义）
   - **最大图集宽度**（流式布局）：设置图集的最大宽度
   - **列数**（网格布局）：设置每行列数
   - **POT对齐**：将图集尺寸向上取整到2的幂次

4. **实时预览**
   - 参数调整后，预览区实时显示排版效果
   - 底部显示图集尺寸、帧数量等信息

5. **导出精灵图**
   - 点击「导出精灵图」按钮
   - 会生成 PNG 格式的精灵图和对应的 JSON 元数据文件

6. **导出 GIF（可选）**
   - 点击「导出 GIF」按钮
   - 设置 GIF 帧率和循环次数
   - 导出 GIF 动画文件
